# Project Documentation — Automation Exercise (Playwright)

## 1) Overview
Automation Exercise - Playwright framework built in **JavaScript** using the **Page Object Model (POM)**. Includes Playwright tests, page objects, and utility helpers with logging and reporting.

## 2) Tech Stack
- **Playwright (Node.js)**
- **CommonJS** modules
- **HTML reporter**
- **Jenkins pipeline** for CI

## 3) Repository Layout

### Top-level
- `package.json` — dependencies and npm scripts
- `playwright.config.js` — Playwright configuration
- `Jenkinsfile` — CI pipeline
- `README.md` — main project documentation
- `TODO.md` — tracking remaining work
- `RESTRUCTURE_SUMMARY.md` — restructure notes

### `pages/` (Page Objects)

## 4) Playwright Configuration (`playwright.config.js`)
Key settings read from the repo:
- `testDir: './tests'`
- `timeout: 60000`
- `fullyParallel: false`
- `reporter: [['html']]`
- Browser: `chromium`
- `headless: false`
- `slowMo: 1000`
- Viewport: `1400x900`
- Screenshots: `only-on-failure`
- Video: `retain-on-failure`
- Trace: `retain-on-failure`
- `outputDir: 'test-results'` (ensures traces/screenshots paths exist)

## 5) npm Scripts (`package.json`)
- `npm test` / `npm run test`: `playwright test`
- `test:headed`: `playwright test --headed`
- `test:single`: `playwright test --grep "@single"`
- `report`: `playwright show-report`
- `debug`: `PWDEBUG=1 playwright test --ui`
- `html-report`: `playwright test --reporter=html`

## 8) CI/CD (Jenkins)
Jenkins pipeline stages from `Jenkinsfile`:
1. Clone repository
2. `npm install`
3. `npx playwright install`
4. `npx playwright test`
5. Post-build: archive artifacts (Playwright report)

## 9) Execution Commands
Run all tests:
```bash
npx playwright test
```

Run headed:
```bash
npx playwright test --headed
```

Open HTML report:
```bash
npx playwright show-report
```
